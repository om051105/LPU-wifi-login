# LPU WiFi Auto-Connect Extension

## Installation
1. Open Chrome/Edge and go to `chrome://extensions`.
2. Enable **Developer mode** (toggle in the top right).
3. Click **Load unpacked**.
4. Select this folder (`LPU wifi login`).

## Usage
1. Click the extension icon in your browser toolbar.
2. Enter your LPU **User ID** and **Password**.
3. Click **Save Credentials**.
4. Valid credentials are now stored safely in your browser.
5. Next time you visit the LPU WiFi login page, it will auto-connect!

## Note
If the extension doesn't fill the fields, the website structure might have changed.
Right-click the page, choose "Inspect", and check the `name` or `id` of the input fields.
Update `content.js` accordingly.
